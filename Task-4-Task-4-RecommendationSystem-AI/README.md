# Task 4 - Recommendation System

## Objective
Build a simple recommendation system that suggests movies based on the user's favorite genre.

## Language
Java

## Features
- Accepts user's favorite genre.
- Recommends movies using if-else statements.
- Simple and beginner-friendly.

## Sample Input
thriller

## Sample Output
Recommended Movies:
- Inception
- Prisoners
- Se7en

## Concepts Used
- Java
- Scanner Class
- if-else statements
- String comparison
