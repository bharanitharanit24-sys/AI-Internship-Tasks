# Task 5 - Face Detection and Recognition

## Objective
Build a simple Java program that simulates face detection and recognition.

## Language
Java

## Features
- Detects a face (simulation).
- Recognizes known persons.
- Displays "Person Not Recognized" for unknown faces.

## Sample Input
Alice

## Sample Output
Face Detected!
Recognized Person: Alice

## Concepts Used
- Java
- Scanner Class
- if-else statements
- String comparison
