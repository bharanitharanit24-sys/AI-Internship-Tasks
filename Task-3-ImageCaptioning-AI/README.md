# Task 3 - Image Captioning AI

## Objective
Build a simple image captioning program that generates captions for images.

## Language
Java

## How It Works
- User enters an image name.
- The program checks the image using if-else conditions.
- It displays a matching caption.

## Sample Input
cat

## Sample Output
Caption: A cute cat is sitting on the floor.

## Concepts Used
- Java
- if-else statements
- User input
- String comparison
